
To configure the game, edit the <HOST> in index.php on line 26 and edit <webSocket> on line 101 in server.php to match.



To run the server:
1. cd to the folder that contain server.php
2. type 'php -q server.php' from the command line.

Note: never close the command prompt windows (if not the server will stop)

To play:
1. put the code to a folder into www folder of apche, or folder that contains codeigniter
2. suppose the url is: connect4/index.php
3. In the browser, try: [address]/connect4/index.php
 e.g. localhost/connect4/index.php
